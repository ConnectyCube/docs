With ConnectyCube code samples - learn how to implement 1-1 and group chat messaging, video calling, enable Push Notifications, authenticate your users via phone SMS verification, store and retrieve file attachments from the cloud and many more features.

These code samples are simple enough that even novice developers will be able to understand them.

Cordova code samples are available at [GitHub repository](https://github.com/ConnectyCube/connectycube-cordova-samples)
